# python大容量字典加速方案

字典规模过大时，哈你函数会出现冲突，即不同的键映射到相同的位置，一般会通过链式哈希来解决。
但性能还是不行。

需要通过字典分片技术，将一个大字典分成多个小字典，每个小字典中哈希冲突的概率会降低，进而提升查找性能。

```python
import cityhash
# 一个大字典转换成多个小字典 分片存储
def big_to_listof_little_dict(big_dict):
    little_dict_num = 1000
    listof_little_dict = [{} for _ in range(little_dict_num)] # 这里分成了1000个小字典，视情况调整
    for key,value in big_dict.items():
        # 计算数字hash
        digital_hash = cityhash.CityHash64(key) # 字符串转hash
        # 计算具体要分片到哪个小dict
        list_index = digital_hash % little_dict_num
        # 存入对应的小字典中 注意小字典中的key还是原来的字符串
        listof_little_dict[list_index][key] = value
     return listof_little_dict
     
# 从分片存储字典序列中获取查询键对应的值
def get_value(list_dict, key):
    little_dict_num = len(list_dict)
    # 计算键所属小字典索引
    digital_hash = cityhash.CityHash64(key)
    list_index = digital_hash % little_dict_num
    # 获取对应小字典中的键对应的值
    return list_dict[list_index][key]

```

# 耗时

测试的方法使用 timeit.Timer 对象，通过设置 setup 来定义一些准备工作代码，通过 stmt 设置需要测试运行时间的代码，通过 timeit 方法来测试性能（可以输入运行的次数，计算平均数）
实际测试时，首先需要进行热身，即先运行一次需要运行的内容，其次如果运行的时间少于1s则需要运行多次算平均。

https://zhuanlan.zhihu.com/p/104440866

```py
# 设置需要测试的代码
timer = timeit.Timer(setup='import numpy as np',
                     stmt='np.zeros((4, 4))')
timer.timeit(10)
# 后续都会用到的测试代码
def bench_workload(workload):
    """Benchmark a workload

    workload: a method that accept a num_repeat argument
    and return its total execution time
    """
    # 需要先预热一波
    workload(1)  # warmup

    # 测试单次的运行时间
    # 如果单词运行时间大于1s则则就运行一次
    time = workload(1)  # the time to run once
    if time > 1: return time

    # 如果单词运行时间少于1s，则运行多次计算平均
    # The number of repeats to measure at least 1 second
    num_repeats = max(int(1.0 / time), 5)
    return workload(num_repeats) / num_repeats

print('time for np.zeros((4, 4)):', bench_workload(timer.timeit))


```

```py
def  convert(filename,out_enc="utf-8"):
    with open(filename, 'rb+') as f :
        content = f.read()
        encode = chardet.detect(content)['encoding']
        if ( encode != out_enc):
            try:
                gbk_content = content.decode(encode)
                uft_byte = bytes(gbk_content, encoding=out_enc)
                f.seek(0)
                f.write(uft_byte)
                print(filename + "\n  src_encodeing:" + encode + "  convert_to_encodeing:" + out_enc)
```

# 1. 利用嵌套循环 打印九九乘法表

思路分析:

    1.打印星星
    2.使用嵌套循环打印星星
    3.将星星替换成乘法口诀公式

> 使用嵌套循环打印星星

```py
row = 1
while row < 10: # 行循环  1,2,...,9打印9行
    # 这一行,需要操作n次 定义循环
    # 列循环:操作的次数
    col = 1
    while col <= row: # 每行的数量 和当前行数相等
        # 不需要换行
        print('*',end='') # 默认会有换行，end=''，不换行
        col += 1 # 列数+1
    # 换行
    print()
    row += 1      # 行数+1
     
```

> 将星星替换成乘法口诀

```py
row = 1
while row < 10:# 行循环  1,2,...,9打印9行
```
